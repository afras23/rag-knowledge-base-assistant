# Operations Security Baseline

## Access control

Production systems require multi-factor authentication for interactive access. Service accounts must rotate credentials every ninety days. Administrative access is granted through a break-glass procedure with time-bound elevation.

## Logging

Security-relevant events must be forwarded to the central SIEM within five minutes. Failed authentication bursts trigger automated account lockout review.

## Incident response

Severity-one incidents require a war room within thirty minutes and customer notification when personal data is impacted.
